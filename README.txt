This repo contains the source, graphics and tools for Toob. The directories here contain :

editor - Level editor, written in C# for Visual Studio 2017
game - Game source, written for Ophis assembler and using Swift as IDE
models - 3D models, use Blender
tools - Various graphics conversion and table generation tools, written in C

As of 02/02/2019, I've abandoned this game, so feel free to do what you like with it!
